Class: MatcapMaterial

zen3d.MatcapMaterial()

new MatcapMaterial()

MatcapMaterial is defined by a MatCap (or Lit Sphere) texture, which encodes the material color and shading. MatcapMaterial does not respond to lights since the matcap image file encodes baked lighting. It will cast a shadow onto an object that receives shadows (and shadow clipping works), but it will not self-shadow or receive shadows.
